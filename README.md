# Sheets JSON Export

Export Google Sheets ranges to LLM-friendly JSON format.

## Development Setup

```bash
# Authenticate with Google Apps Script
clasp login

# Push code to Apps Script
clasp push

# Open in Apps Script editor
clasp open
```

## Deployment

Auto-deploys via GitHub Actions on push to main.

Manual deployment:
```bash
clasp push
```
